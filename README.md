## What I've Built

| Project | What it actually does | Stack |
|---|---|---|
| [AI Compliance Assistant](https://github.com/tanmayi123/ai-compliance-assistant) | Multi-agent RAG system covering HIPAA, GDPR, EU AI Act, FINRA and CCPA. Intelligent query routing sends each question to the right compliance agent, with Pinecone vector search, live web fallback, source-cited responses, and PDF report export | LangChain · LangGraph · Pinecone · Tavily · Python |
| [Momento: Reader Matching Agent](https://github.com/tanmayi123/Moment) | Agent pipeline where a decomposition agent extracts structured claims from reader profiles and a compatibility agent scores similarity across reasoning depth, emotional alignment, and interpretive overlap. Production data pipeline via Airflow, TFDV, and DVC | Gemini API · Airflow · TFDV · DVC · MLOps · GCP |
| [Customer Segmentation Platform](https://github.com/tanmayi123/Customer-Segmentation) | End-to-end platform on 541K UCI transactions: RFM segmentation via K-Means, three supervised ML models (churn ROC-AUC 0.727, CLV regression, next-purchase propensity 0.713), and Gemini-generated executive narratives per segment. Deployed as a Streamlit app via Docker | Python · scikit-learn · Gemini API · Docker · Streamlit |
| [Peer-to-Peer Lending Analytics](https://github.com/tanmayi123/Peer-2-Peer-Lending-System) | Full-stack analytics platform on an 18-table PostgreSQL schema hosted on Supabase. Six SQL analytics views cover borrower risk, lender performance, repayment health, and platform overview. Live Streamlit dashboard plus 4-sheet Tableau viz with LOD expressions and MoM calculations | PostgreSQL · Supabase · Streamlit · Tableau · Plotly |
| [Early Warning System](https://github.com/tanmayi123/Early-Warning-System-for-Detecting-Critical-Deterioration-in-Hospitalized-Patients) | Clinical ML system predicting patient deterioration severity across a 5-level ordinal scale on 9,105 SUPPORT2 records. XGBoost selected via Quadratic Weighted Kappa (0.87, "almost perfect" agreement), with KNN and Random Forest imputation and three-layer feature selection | XGBoost · scikit-learn · Streamlit · Python |
